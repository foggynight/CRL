# rtb

**rtb** is a zero-dependency catch-all toolkit for C/C++ focused on
reusability and performance.

## Directories

- `docs`: Documentation
- `src`: Source code
- `test`: Unit tests

## Components

- `rtb_list.h`: Generic linked list implementations
- `rtb_log.h`: Macros for printing to the stderr stream
